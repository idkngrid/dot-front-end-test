## Pré Requisitos 
Antes de começar, é preciso ter ter o npm e o [node](https://nodejs.org/en/) instalados.

## Instalação

```bash
  # Clone o repositório
  $ git clone https://github.com/idkngrid/dot-front-end-test.git
  # Acesse a pasta
  $ cd dot-front-end-test
  # Instale as dependências 
  $ npm install
  # Execute a aplicação
  $ npm start
```
